# ArcGIS Enterprise Administration Professional 2025 (EAEP_2025) Practice Test

Practice exam materials aligned to the **ArcGIS Enterprise Administration Professional 2025** exam (EAEP_2025), based on the official Exam Information Guide ([EAEP2025_EIG_JAN2026.pdf](https://community.esri.com/t5/esri-technical-certification-exams/eaep-2025-eig/ba-p/1595261)) and the same delivery pattern as `arcgis_pro_foundation_2025`. Single-page HTML; no server required when using the standalone build.

**Developer:** Kyros

## Place the EIG PDF locally

Download **EAEP2025_EIG_JAN2026.pdf** from the [Esri Community EAEP 2025 EIG article](https://community.esri.com/t5/esri-technical-certification-exams/eaep-2025-eig/ba-p/1595261) and save it in this folder as `EAEP2025_EIG_JAN2026.pdf` so it matches `exam-spec.json` and your local reference.

## Contents

| File | Description |
|------|-------------|
| `EAEP2025_EIG_JAN2026.pdf` | Official Esri Exam Information Guide (place locally; see above). |
| `exam-spec.json` | Machine-readable exam metadata and domain breakdown from the EIG. |
| `questions.json` | Question bank (500 unique stems) mapped to exam domains; first 10 from CertFun samples. |
| `practice-test.html` | Practice test UI: loads `questions.json` (or file picker when opened as `file://`). |
| `practice-test-standalone.html` | Single-file build with the bank embedded; open directly with no server. Generated by `build_standalone.py`. |
| `build_standalone.py` | Embeds `questions.json` into the HTML. Run after updating questions. |
| `serve.py` | Optional local HTTP server for `practice-test.html`. |
| `generate_questions_cursor.py` | Cursor prompt + merge workflow for new questions. |
| `prompt_for_cursor_questions.txt` | Last-generated Cursor prompt (regenerate with `python generate_questions_cursor.py prompt`). |
| `generate_questions.py` | Template-based generator for additional items. |
| `emit_bulk_questions.py` | Emit many unique stems quickly (combines intro phrases with templates); e.g. `python emit_bulk_questions.py -n 400 > new_questions.json` then `merge`. |
| `generate_help.py` | Short usage summary. |
| `PRD.md` | Product requirements for this practice test. |

## Exam at a glance (from EIG)

- **Questions:** 75  
- **Duration:** 1 hour 45 minutes (105 minutes)  
- **Format:** Multiple choice  
- **Domains and weight:**  
  - Deploy ArcGIS Enterprise **34%**  
  - Troubleshoot ArcGIS Enterprise **18%**  
  - Maintain and Support ArcGIS Enterprise **22%**  
  - Manage Content and Users in ArcGIS Enterprise **26%**

Official exam summary: [ArcGIS Enterprise Administration Professional 2025 – Esri Academy](https://www.esri.com/training/catalog/67e2da62d6844f1703724c6f/arcgis-enterprise-administration-professional-2025/).

## Question bank note

Questions **1–10** in `questions.json` are the free sample items from [CertFun’s EAEP_2025 sample questions](https://www.certfun.com/esri/esri-arcgis-enterprise-administration-professional-eaep-2025-certification-sample-questions), with a **fourth distractor** added so every item matches this app’s four-option format. Correct answers match CertFun’s published key. The remaining items are original practice stems aligned to the EIG domains. The bank is **deduplicated by question text** (first occurrence kept). To add a large batch without hand-pasting Cursor output, use `emit_bulk_questions.py`. For fully new stems from a model, use `prompt_for_cursor_questions.txt` (regenerate with `python generate_questions_cursor.py prompt --count 400`).

## How to use the practice test

**Option A – Standalone**  
Open `practice-test-standalone.html` in a browser. Regenerate after bank changes: `python build_standalone.py`.

**Option B – Two-file**  
1. Open `practice-test.html`.  
2. Load the bank: run `python serve.py` and use the URL shown, or use “Load from your computer” if you opened the file via `file://`.  
3. Each run draws **75** random questions (or all bank items if fewer than 75).

## Adding more questions

Same workflow as ArcGIS Pro Foundation: run `python generate_questions_cursor.py wizard` or `prompt` / `merge`, or `python generate_questions.py` and merge the output. See `generate_help.py`.
